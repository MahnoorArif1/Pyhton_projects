# Pong Game with Python Turtle

A simple two-player Pong game built using Python's `turtle` module. Players control paddles to hit the ball and score points.

## Features
- Two-player gameplay
  - Player 1: Up/Down arrow keys
  - Player 2: D/C keys
- Score tracking
- Ball collision with paddles and walls
- Game over display

## Files
- `main.py` – Runs the game loop and handles input and collisions
- `bridge.py` – Draws the center dividing line
- `paddle.py` – Defines paddle movement
- `ball.py` – Handles ball movement and collision detection
- `score.py` – Tracks and displays scores

## How to Play
1. Run `main.py`.
2. Player 1 moves with **Up** and **Down** keys.
3. Player 2 moves with **D** (up) and **C** (down) keys.
4. Hit the ball with your paddle to score points.
5. The game ends when a player reaches the edge.

## Requirements
- Python 3.x
- Standard library only (`turtle`, `time`)
