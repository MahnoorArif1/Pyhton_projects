# US States Guessing Game

A fun interactive game built with Python, Turtle, and Pandas where players guess all 50 US states on a blank map.

## Features

- Guess US states by typing their names.
- Hint feature provides the first letter of a remaining state.
- Pop-up messages for correct guesses, incorrect guesses, duplicate guesses, hints, and exit.
- Clean and visual interface using Turtle graphics.
